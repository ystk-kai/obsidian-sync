使用方法

ObsidianのLiveSyncプラグインで以下のURIを設定してください：

Remote Database URI: http://サーバーアドレス/db

重要な注意事項

  • URI形式: 「http://」または「https://」を使用してください。
  • 接続テスト: このページでの接続ステータスは診断目的のみです。
  • プラグイン設定: LiveSyncプラグインの設定でRemote Database URIに上記のURIを設定してください。
